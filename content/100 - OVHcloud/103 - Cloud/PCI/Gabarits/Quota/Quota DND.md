Bonjour XXXX,

Vos ressources Public Cloud (RAM, CPU, espace disque, instances, etc.) sont limitées par défaut. Ces quotas s’ajustent automatiquement en fonction de votre utilisation.

Je vous suggère de vérifier vos quotas, consultez-les dans votre espace client OVHcloud : https://manager.eu.ovhcloud.com/#/hub/

-Rubrique "Public Cloud"
-Onglet "Quota and regions"

Vous pouvez augmenter vos quotas manuellement, afin d'anticiper vos besoins, ajoutez un crédit d’utilisation (valable 12 mois). Ce crédit servira à payer vos prochaines factures Public Cloud.

Suivez ce guide ci-dessous et consultez le tableau "Augmenter manuellement votre quota" pour connaître le montant nécessaire.

Lien : https://help.ovhcloud.com/csm/fr-public-cloud-compute-increase-quota?id=kb_article_view&sysparm_article=KB0050857

Exemple : Pour une instance "b3-128", un quota de "20 VMs" est suffisant.

-Centre d’aide OVHcloud : https://help.ovhcloud.com/csm/fr-home
