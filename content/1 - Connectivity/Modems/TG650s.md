## TG650s

##### Bridge

Bonjour Monsieur,

Comme convenu lors de notre échange téléphonique, je vous envoie la procédure afin d'activer le mode bridge sur l'interface locale du modem.

Veuillez cliquer sur le lien ci-dessous pour télécharger le fichier de configuration modem bridge.

Lien: http://files.isp.ovh.net/650s/bridge/

Ensuite, vous devez insérer ce fichier dans le modem directement en suivant la procédure ci-dessous.

1 - Enregistrez le fichier "650spppbridge.ini" grâce à un clique droit puis "Enregistrer la cible du lien sous"
2 - Rendez vous sur l'interface locale du modem via ce lien http://192.168.1.254
3 - Cliquez sur "Media Access Gateway" ou "Technicolor Gateway"
4 - Cliquez sur "Configuration" dans le menu déroulant
5 - Cliquez sur "Enregistrer ou Restaurer la configuration"
6 - Cliquez sur "Restaurer avec un fichier de configuration"
7 - Sélectionnez le fichier "650spppbridge.ini"
8 - Cliquez sur "Restaurer"
9 - Le modem redémarre une fois la barre de chargement terminée.

Le modem est en bridge, vous devez maintenant établir la session de connexion avec vos identifiants de connexion PPPOE. Vous avez dû les recevoir par mail suite à l'activation du mode bridge sur cette adresse : MAIL

##### PPPOE

Bonjour Monsieur,

Veuillez suivre la procédure ci-dessous pour établir la session PPPOE sur votre modem.

1 - Rendez vous sur l'interface locale du modem via ce lien http://192.168.1.254
2 - Cliquez sur "Broadband connexion" ou "Connexion Large bande"
3 - Cliquez sur "Service internet" dans le menu déroulant
4 - Cliquez sur "Afficher plus" situé en dessous de "@ Internet"
5 - Cliquez sur "Déconnecter" en haut à droite
6 - Rentrez le nom d'utilisateur et le mot de passe PPPOE
7 - Cliquez sur "Connecter"
