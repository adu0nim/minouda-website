## SIP ALG
Bonjour Monsieur,

Veuillez trouver ci-dessous la procédure pour vérifier si la fonctionnalité SIP ALG est activé sur le modem.

-Pour récupérer l’état actuel du SIP ALG, utiliser l'API :
https://api.ovh.com/console/#/xdsl/%7BserviceName%7D/modem/sipAlg#GET

Si la fonction est désactivée, je vous invite à l'activer en utilisant le lien ci-dessous.
-Pour l'activer ou le désactiver :
https://api.ovh.com/console/#/xdsl/%7BserviceName%7D/modem/sipAlg#POST
