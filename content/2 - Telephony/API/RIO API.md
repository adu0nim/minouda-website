## RIO API

Bonjour,

Voici la procédure pour récupérer les codes RIO par API :

1 - Connectez-vous sur https://eu.api.ovh.com/console-old/ puis cliquez sur "Login" en haut à droite
2 - Rendez-vous sur cette API : https://eu.api.ovh.com/console-old/#/telephony/%7BbillingAccount%7D/service/%7BserviceName%7D~GET

Pour les paramètres :
- billingAccount = Groupe téléphonie 
- serviceName = Numéro de l'alias ou de la ligne SIP au format international
