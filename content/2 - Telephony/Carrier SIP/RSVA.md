## RSVA
From Rejeté RSVA:

Le NUMERO est un numéro RSVA, et comme indiqué dans votre contrat CarrierSIP, il n'est pas autorisé de présenter ce type de numéro lors d'appels sortants.

Pour rappel :
Lors de l'établissement d'un appel, ce dernier doit être émis depuis l'Union Européenne.
Pour chaque pays concerné, les appels doivent être de format :
* Ligne fixe
* Mobile
* Numéro personnel
Les appels ne répondant pas à ces contraintes, tels que les VAS (Value-Added Services), seront refusés avec un code d'erreur 403 (forbidden).
