## Yealink
>Lien de prov : yl.prov.voip.ovh.net

#### Dépannage P&P + Reset 
Bonjour Monsieur,

Vous allez devoir réinitialiser votre téléphone en suivant la procédure ci-dessous.

1 - Restez appuyé sur le bouton "OK" du téléphone durant 10 secondes.
2 - Confirmez en appuyant de nouveau sur "OK", quand la fenêtre de confirmation apparaît.
3 - Le téléphone redémarre.

Veuillez suivre la procédure ci-dessous afin d'envoyer le fichier de configuration au téléphone.

1 - Authentifiez-vous sur votre espace client via ce lien :
https://www.ovhtelecom.fr/manager/#/.
2 - Cliquez sur "Télécom" dans la barre en haut à gauche
3 - Cliquez sur "Téléphonie" dans la colonne à votre gauche
4 - Sélectionnez le groupe téléphonie concerné
5 - Sélectionnez votre ligne SIP en dessous de votre groupe téléphonie
6 - Cliquez sur "Assistance" sur la page principale.
7 - Cliquez sur "Dépannage Plug & Phone" sur la page principale.
8 - Cochez "Autre - reconfigurer automatiquement votre équipement (mode expert)"
9 - Entrez l'Adresse IP publique de votre modem que vous pouvez récupérer en allant sur http://ifconfig.ovh/
Il faut être sur le même réseau que le téléphone.

#### Transfert

Le transfert accompagné vous permet d'avoir le destinataire du transfert avant de lui passer l'appel. Il peut ainsi accepter ou refuser le transfert, ou encore, si vous ne parvenez pas à le joindre, reprendre votre correspondant. 

1) Vous devez être en ligne avec votre correspondant. 
2) Appuyez sur la touche "Transfert" et composez le numéro, ou, dans le carnet d'adresses, le destinataire du transfert.
3) Appuyez sur la touche "OK" du téléphone. Le destinataire du transfert accepte l'appel. 
4) Appuyez sur la touche "Transfert" pour envoyer l'appel vers le destinataire

##### Provisionning

Monsieur,

Merci d'avoir effectué les tests. Pouvez-vous renseigner le lien de provisionning dans l'interface web du téléphone Yealink s'il vous plaît ? 

Voici la procédure : 

1 - Ouvrez l'interface web de votre téléphone IP Yealink et connectez-vous (login et mot de passe par défaut sont admin/admin).
2 - Naviguez dans “Settings” >> “Auto Provision”.
3 - Entrez votre URL de “lien de provisioning” dans la section “Server URL”.
4 - Appuyez sur “Confirm” et ensuite “Autoprovision now”. 

> Le lien de provisionning est : http://yl.prov.voip.ovh.net/device

#### Ligne Interdite

Bonjour,

Lorsque le message "Ligne interdite" s'affiche sur votre combiné, cela signifie que le combiné n'est pas correctement associé à la ligne OVH afin de pouvoir passer des appels. Je vous invite à vérifier via l'interface Web du téléphone que les combinés sont correctement associés pour les appels entrants et sortants. Il faut commencer par récupérer l'adresse IP locale de la base. Pour cela, appuyez brièvement sur le bouton de la base, l'adresse IP s'affichera sur les combinés.
Entrez cette adresse IP dans un navigateur (Firefox, Chrome, Safari) depuis un poste sur le même réseau que le téléphone. Renseignez l'identifiant "user" et le mot de passe "XXXX" (vous pouvez retrouver ce mot de passe dans les informations générales de votre ligne sur l'espace client OVH).

Une fois sur l'interface web du Yealink :

1 - Rendez-vous dans "Compte" puis "Number assignment".
2 - Cochez les cases Incoming lines et outgoing lines pour chaque combiné (Handset) dans la première colonne.
3 - Cliquez ensuite en bas sur "Confirmer".

#### DECT
Bonjour Monsieur,

Vous allez devoir réinitialiser votre téléphone en suivant la procédure ci-dessous.

1 - Rendez vous dans le menu du combiné en appuyant sur la touche "OK".
2 - Rendez vous dans le menu configuration représentée par une roue crantée.
3 - Allez dans "Système" puis "Reset base" (le code PIN par défaut est 0000).

Veuillez suivre la procédure ci-dessous afin d'envoyer le fichier de configuration au téléphone.

1 - Authentifiez-vous sur votre espace client via ce lien :
https://www.ovhtelecom.fr/manager/#/.
2 - Cliquez sur "Télécom" dans la barre en haut à gauche.
3 - Cliquez sur "Téléphonie" dans la colonne à votre gauche.
4 - Sélectionnez le groupe téléphonie concerné
5 - Sélectionnez votre ligne SIP en dessous de votre groupe téléphonie
6 - Cliquez sur "Assistance" sur la page principale.
7 - Cliquez sur "Dépannage Plug & Phone" sur la page principale.
8 - Cochez "Autre - reconfigurer automatiquement votre équipement (mode expert)"
9 - Entrez l'Adresse IP publique de votre modem que vous pouvez récupérer en allant sur http://ifconfig.ovh/
Il faut être sur le même réseau que le téléphone.

Les combinés doivent s'associer à la base automatiquement, si ils ne le font pas vous pouvez suivre la procédure ci-dessous.

1 - Rendez vous dans le menu du combiné en appuyant sur la touche "OK".
2 - Rendez vous dans le menu configuration représentée par une roue crantée.
3 - Allez dans "Enregistrement" puis "Inscrire combiné.
